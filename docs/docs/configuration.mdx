### Step 1: Access Settings

1. **Go to WordPress Admin ‚Üí Pika Finance**
2. **Click the "Settings" icon** (gear icon) in the top right
3. **Or navigate to `yourdomain.com/wp-admin/admin.php?page=pika-settings`**

### Step 2: Set Up Your Gemini API Key

To enable Pika's AI-powered features, you'll need to connect your Gemini API key:

1. **Generate an API Key:**  
   - Visit [Gemini Studio](https://aistudio.google.com/app/apikey) and sign in with your Google account.
   - Follow the instructions to create a new API key for your project.

2. **Add the API Key to Pika:**  
   - In the Pika Finance settings page, locate the **"Gemini API Key"** field.
   - Paste your copied API key into this field.

3. **Save Your Settings:**  
   - Click **"Save"** or **"Update Settings"** to apply the changes.

:::tip
Keep your API key secure and never share it publicly. If you believe your key has been compromised, generate a new one in Gemini Studio.
:::
